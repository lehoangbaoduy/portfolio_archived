Credit Card Approval Prediction

Sep 2020 - Nov 2020

Numerous applications for credit cards are submitted to commercial banks. Many of them are turned down for a variety of reasons, such as excessive credit record queries, high loan amounts, and low income levels. These analyses must be done manually, which is tedious, error-prone, and time-consuming (and time is money!). Fortunately, machine learning can automate this process, and presently, almost every commercial bank does. In this project, we will use machine learning techniques to create an automatic credit card acceptance predictor, much as actual banks use.

The project begins with the dataset being loaded into our Jupyter notebook. The dataset is added to a pandas dataframe called df. The dataset contributor has anonymised the feature names in order to maintain the data's confidentiality. The letters A through P are given feature names. We use df.head() to print the top five rows of the dataset so that we can immediately review it.

In order to forecast if a credit card would be approved, we tested five different categorization models. The graph below provides a summary of the models' train and test accuracy. The Logistic Regression classifier provided us with the highest test data accuracy (88%) in the results. The lack of overfitting and underfitting is indicated by the little discrepancy between train and test accuracy scores.
